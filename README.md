# Diákzóna Project

## Project info

Interaktív tanulási platform általános iskolások számára.

## Technologies Used

- Vite
- TypeScript
- React (with React Router)
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
